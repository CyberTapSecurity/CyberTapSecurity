NCL Forensics Writeup

This writeup details the forensics challenges from the National Cyber League (NCL), where I utilized various forensic tools to investigate and recover digital evidence related to suspicious activities.

Challenge Overview

In this forensics challenge, the goal was to recover deleted files, analyze logs, and extract crucial data from a compromised machine. The challenge involved parsing through evidence files and identifying traces of malicious activity, using industry-standard tools like Autopsy, FTK Imager, and Sleuth Kit.

Tools Used

  • Autopsy: A graphical interface to The Sleuth Kit (TSK), for analyzing disk images.
  • FTK Imager: For acquiring disk images and extracting data.
  • Sleuth Kit: A suite of command-line tools for digital forensics.
  • Wireshark: For packet analysis in network traffic investigation.

Steps Taken

  1. Initial Analysis: Loaded the disk image into Autopsy to begin an initial inspection of the file system.
  2. File Carving: Used FTK Imager and Autopsy to locate and recover deleted files that could contain evidence.
  3. Log Analysis: Examined system logs and application logs for signs of intrusion or suspicious behavior.
  4. Network Traffic Analysis: Captured network traffic using Wireshark to identify potential malicious network connections and data exfiltration.

Findings

The analysis revealed several key pieces of evidence, including:

  • Deleted files with suspicious names that were recovered and found to contain malware executables.
  • Log entries showing multiple failed login attempts from an IP address associated with a known botnet.
  • Network traffic showing an abnormal amount of data being sent to an external IP, confirming data exfiltration activities.

Conclusion

The forensics challenge showcased how essential digital forensics is in uncovering hidden data and malicious activity. By using the right tools and techniques, we were able to recover critical evidence and understand the full scope of the breach. The findings were crucial for building the case of the attack and understanding the attacker's methodology.
